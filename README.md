# Tugas Koneksi PHP & MySQL

Project sederhana dibuat untuk memenuhi tugas pemrograman web, mendemonstrasikan cara menghubungkan aplikasi PHP (Native) dengan database MySQL dan menampilkan data ke dalam bentuk tabel HTML.

# Fitur Utama
- Menghubungkan script PHP dengan database MySQL menggunakan fungsi `mysqli`.
- Mengambil data (*fetch data*) dari tabel database dan menampilkannya di halaman web.

# Teknologi yang Digunakan
- Bahasa Pemrograman: PHP
- Database: MySQL
- Environment: Laragon (Localhost)
